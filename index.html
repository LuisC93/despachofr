// GitHub — datos.json en tu mismo repositorio

¿Cómo funciona?

1

Esta app lee y escribe un archivo datos.json directamente en tu repositorio de GitHub usando la GitHub API. Todos los que abran el GitHub Pages leen el mismo archivo.

2

Ve a github.com/settings/tokens/new → crea un token clásico con permiso repo → cópialo.

3

Pega tu usuario, nombre del repositorio y el token abajo. El token se guarda solo en este dispositivo y nunca sale de tu navegador.
